Touch-WSGI

Sublime-Text 2 plugin to touch all WSGI files in currently open folders, in the current window, so you don't have to.

This is available in the Sublime Text Package Control.

Searches the Currently open folders (not subfolders) for files with the extension of wsgi and touches the file.
